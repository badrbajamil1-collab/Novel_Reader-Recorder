# Novel_Reader-Recorder

This is a fun project I made — the goal is to create something similar to an audiobook but for web novels.

![screenshot](https://github.com/user-attachments/assets/0b1a395c-8ccd-4eb5-8922-bad631a83b7c)

It works with **novelfire.net** and **novelphoenix.com**, but you can easily add different websites by asking an AI!

**How to add a new website:**
1. Open the chapter page you want to support
2. Press `Ctrl + U` to view the page source
3. Copy the HTML code
4. Paste it to an AI and ask it to add support for that site
5. It should just work!

## Installation

1. Install the **Tampermonkey** extension from your browser's extension store
2. Open Tampermonkey and click **"Create a new script"**
3. Copy the script code and paste it in, or upload it directly to Tampermonkey
4. Save it — and you're done, it just works!
